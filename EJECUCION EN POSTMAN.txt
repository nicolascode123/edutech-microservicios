# EduTech Microservicios

## Microservicios incluidos:

- ms-usuarios: Gestión de usuarios.
- ms-autenticacion: Registro y login.
- ms-cursos: Administración de cursos.
- ms-notificaciones: Envío de notificaciones.

## Requisitos

- Java JDK 17
- Visual Studio Code con extensiones Java
- Postman



## Cómo ejecutar cada microservicio

Abre una terminal distinta para cada uno y corre:

### ms-usuarios (Puerto 8081)
```bash
cd ms-usuarios
./mvnw spring-boot:run
Endpoints:

POST http://localhost:8081/api/usuarios (crear usuario)

ms-autenticacion (Puerto 8082)
bash
Copiar
Editar
cd ms-autenticacion
./mvnw spring-boot:run
Endpoints:

POST http://localhost:8082/auth/register (registrar usuario)

POST http://localhost:8082/auth/login (login)

ms-cursos (Puerto 8083)
bash
Copiar
Editar
cd ms-cursos
./mvnw spring-boot:run
Endpoints:

POST http://localhost:8083/cursos (crear curso)

bash
